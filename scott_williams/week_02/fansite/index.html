Who is Sophie Wilson?

Sophie Mary Wilson, born 1957, is an English computer scientist. The processor that she designed, the ARM, has become one of the most popular IP cores (or chips) in the world. IP cores are the building blocks within integrated circuits.

ARM stands for Advanced Reduced instruction set computing Machine. The ARM is so successful because it was one of the very first low power microprocessors.

It is claimed that Sophie's ARM processor is found in over 35 billion devices, including modern mobile phones and digital cameras.

Something Joel does not know about Sophie

Sophie believes that hardware is no longer what is holding back processor speed. Since 2006, the lack of speed in computation ability is due to scalar programming languages (C, Ruby, Javascript) that are a poor fit for parallel hardware. Sophie believes we should all start programming in OpenCL to take advantage of parallel hardware.
